Gluax — Secure P2P Messaging +

A serverless, metadata-isolated messenger built to close the gap between "encrypted" and actually private.

  • Tor V3 ephemeral hidden services for anonymised routing — no persistent IP exposure, no central server to subpoena.
  • Double Ratchet end-to-end encryption (the Signal Protocol) for forward-secret message delivery.
  • Argon2id + AES-256-GCM local key vault, so even a stolen device doesn't hand over the keys.
  • Verified with live Wireshark traffic analysis across four security tests, all passing, presented as a full report to class.

07 — Leadership
BiSecT — Binary Security Team.

Founder & Team Lead, Emerson University Multan — 2024 to present

BiSecT exists because reading about offensive security and doing it are two different skills. I built a structured training roadmap so members practice both, starting from the ground up rather than jumping straight to exploits.

STAGE 01
Linux Fundamentals

Mastering core CLI utilities, privilege management, file systems, bash scripting, and process execution.

STAGE 02
Reconnaissance

Passive/active OSINT, solving Bandit OverTheWire challenges, and mapping structural layouts.

STAGE 03
Enumeration

Deep-diving service scanning, banners, protocol analysis (SSH, HTTP, SMB), and vulnerability mapping.

STAGE 04
Exploitation

Guided hands-on attacks using Metasploit, manual payload construction, and understanding memory vectors.

STAGE 05
Reporting

Translating root-causes, drafting remediation plans, and practicing responsible mitigation paths.
